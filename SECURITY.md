# Security

Please report security issues privately to the project maintainers.

## Scope

Security-sensitive areas include:

- Voice cloning consent bypasses.
- Hidden network calls.
- Unsafe runner process execution.
- Path traversal in local storage or output routes.
- Model download integrity and license metadata.
- Leaking audio, transcripts, datasets, or logs.

Takokit is local-first. Any feature that sends user data outside the machine must be explicit, visible, and documented.
